# 《AI 宗门模拟器》十年大考弹窗美术需求

版本：v0.1  
日期：2026-06-22  
状态：待美术确认  
适用范围：玩家完成第十年谕令结算后的宗门评分弹窗

## 1. 需求目标

十年大考弹窗是首个十回合周期的结局揭榜，需要同时完成三件事：

1. 给玩家明确的十年阶段结束反馈。
2. 让最终评级、总分和四轴成绩一眼可读。
3. 保留宗门案牍、仙盟敕榜的世界观质感，不做成普通网页数据弹窗。

关键词：敕榜、考卷、朱印、肃穆、阶段荣誉、宗门案牍。

避免：庆典烟花、金光爆闪、紫蓝科技感、现代排行榜、营销海报式大标题。

## 2. 使用场景

### 2.1 自动触发

玩家完成第十年谕令，数值结算与第十年年报成功写入存档后，弹窗自动出现。

触发依据：

```text
state.year === 11
report.rating 存在
本轮大考弹窗尚未自动展示
```

弹窗出现前必须先完成存档，关闭弹窗不能导致评级或年报丢失。

### 2.2 再次查看

宗门史书中提供“查看大考结果”入口，重新打开时直接显示展开后的完整考卷，不重复播放首轮揭榜节奏。

## 3. 交互结构

采用“两阶段揭榜”。弹窗外框尺寸保持稳定，仅内部内容切换，避免展开时整体跳动。

### 3.1 第一阶段：揭榜

首屏只强调最终结果：

```text
仙盟敕榜
十年宗门大考

一方大宗
78
总评

展开考卷
```

信息优先级：

1. 最终评级。
2. 总分。
3. “十年宗门大考”身份说明。
4. 展开操作。

首屏不同时展示四轴、关键转折和长总结，避免信息一次性倾倒。

### 3.2 第二阶段：完整考卷

展开后展示：

```text
最终评级 + 总分
人 / 财 / 物 / 势四轴评分
最强项与最弱项考语
三条关键转折
阶段说明
操作按钮
```

底部操作：

- 主按钮：`查看十年史书`
- 次按钮：`继续执掌`
- 弱操作：`重开十年`

“重开十年”必须二次确认，不能与主按钮使用同等视觉权重。

## 4. 整体构图

### 4.1 遮罩

- 使用深墨棕半透明遮罩，不使用纯黑。
- 建议颜色：`rgba(24, 15, 10, 0.68)`。
- 可使用 1-2px 轻微背景模糊，但主界面卷轴轮廓仍应可辨认。
- 弹窗打开后，背景不可点击和滚动。

### 4.2 主体敕榜

- 形态：纵向敕榜或正式考卷，不使用现代白色矩形卡片。
- 圆角不超过 8px，以纸张边缘和压痕代替大圆角。
- 纸面保持浅黄、灰白、旧墨三种层次，不能全屏泛黄。
- 四角可有克制的回纹、仙盟印记或装订痕迹。
- 不使用卡片套卡片。四轴成绩直接排在考卷纸面上，用分隔线和刻度组织。

### 4.3 桌面尺寸

- 建议宽度：`720-780px`。
- 最大宽度：`min(780px, 88vw)`。
- 最大高度：`86vh`。
- 首屏和展开态使用同一外框尺寸。
- 内容超过高度时，仅考卷正文内部滚动，标题和底部操作保持可见。

### 4.4 移动端尺寸

- 视口小于 720px 时改为近全屏纵向考卷。
- 左右安全边距：12px。
- 上下安全边距：16px。
- 最大高度：`calc(100dvh - 32px)`。
- 四轴评分改为单列或 2×2，不允许文字与数字挤压换行。

## 5. 视觉层级

### 5.1 顶部

- 小字：`仙盟敕榜`
- 主标题：`十年宗门大考`
- 可配一条细墨线或双线回纹。
- 不重复显示宗门案牍、庚子年年报等无关标题。

### 5.2 最终评级

- 评级是第一视觉中心，例如：`一方大宗`。
- 总分是第二视觉中心，例如：`78`。
- 评级可配朱印，但印章不能遮挡字形。
- 评级与总分保持固定区域，长评级名称不能改变布局高度。

建议评级色：

| 评级 | 主色建议 |
| --- | --- |
| 覆灭 | 暗灰褐、残墨 |
| 勉强立足 | 土黄、旧墨 |
| 小有名望 | 青灰、淡金 |
| 一方大宗 | 深青、朱印 |
| 仙门新星 | 玉青、金墨 |

颜色只用于强调，不允许整张弹窗被单一颜色覆盖。

### 5.3 四轴评分

四轴顺序固定：人、财、物、势。

每轴包含：

- 轴名。
- 分数。
- 横向刻度条。
- 一句不超过 18 字的考语。

刻度颜色：

- 65 分及以上：青绿色。
- 45-64 分：沉金色。
- 44 分及以下：朱红色。

刻度条必须有固定宽度和高度，分数变化不能造成布局跳动。

### 5.4 关键转折

- 最多展示三条。
- 使用竖向短时间线或三行案牍记录。
- 不使用三个独立浮卡。
- 每条最多两行，超出使用省略号。
- 年份或序号使用小朱印、墨点或短线标记。

### 5.5 阶段说明

固定说明：

```text
十年大考已定，但宗门仍可继续经营。
```

该说明使用次级文字，不抢最终评级视觉层级。

## 6. 字体与排版

- 标题可沿用项目现有中文衬线或宋体风格。
- 评级字重应明显高于正文。
- 总分使用易读数字字体，不使用难辨识的书法数字。
- 正文最小字号：桌面 14px，移动端 13px。
- 字距固定为 0，不使用负字距。
- 行高：正文建议 1.5-1.7。
- 所有文本必须在容器内完整显示，长标题最多两行。

建议字号：

| 元素 | 桌面 | 移动端 |
| --- | --- | --- |
| 仙盟敕榜 | 13-14px | 12-13px |
| 十年宗门大考 | 24-28px | 21-24px |
| 最终评级 | 34-42px | 30-36px |
| 总分 | 52-64px | 46-56px |
| 四轴分数 | 20-24px | 18-22px |
| 正文 | 14-16px | 13-15px |

## 7. 材质与装饰

允许使用：

- 真实纸张纤维和轻微折痕。
- 朱砂印章、墨线、细回纹。
- 少量金墨用于高评级。
- 敕榜顶部的简洁装订或悬挂结构。

禁止使用：

- 大面积渐变光晕、光球、粒子雨。
- 模糊不可辨的背景图。
- 现代玻璃拟态、霓虹边框。
- 大面积紫色、深蓝色或单一棕黄色。
- 纯装饰 SVG 插画替代主体信息。

## 8. 动效需求

动效总时长控制在 700ms 内，不阻碍玩家操作。

### 8.1 自动揭榜

1. 遮罩在 160ms 内淡入。
2. 敕榜在 220ms 内由轻微下移和缩放恢复到位。
3. 最终评级与总分在 240ms 内显现。
4. 朱印可在最后 160ms 落下，伴随轻微缩放，不做屏幕震动。

### 8.2 展开考卷

- 内容使用淡入和小幅上移。
- 四轴刻度条从 0 填充到目标分数，时长 350-500ms。
- 用户开启“减少动态效果”时，取消位移、缩放和刻度动画，只保留即时切换。

## 9. 状态需求

美术和前端需要覆盖以下状态：

1. 正常评级：五档评级均可适配。
2. 最低评级：避免使用过度夸张的失败血色效果。
3. 最高评级：允许增加少量金墨，但不改变弹窗结构。
4. 长考评：正文内部滚动，按钮始终可用。
5. 无关键转折：显示“史册未录得显著转折”。
6. 再次查看：直接进入完整考卷，不播放首轮揭榜。
7. 移动端：按钮纵向排列，关闭操作始终可见。

## 10. 美术资源交付

优先复用现有卷轴材质，新增资源控制在以下范围：

| 资源 | 建议格式 | 建议尺寸 | 透明背景 |
| --- | --- | --- | --- |
| 大考敕榜底图 | PNG/WebP | 1560×1760 | 是 |
| 仙盟朱印 | PNG/WebP | 320×320 | 是 |
| 高评级金墨装饰 | PNG/WebP | 800×180 | 是 |
| 低评级残墨装饰 | PNG/WebP | 800×180 | 是 |

交付要求：

- 资源边缘保留安全区，不把文字直接画进底图。
- 底图中心区域必须适合动态文字和刻度条覆盖。
- 2× 分辨率交付，前端按 CSS 尺寸缩放。
- PNG 需清理无意义透明边距。
- 同时提供压缩后的 WebP 版本，单张建议小于 500KB。

## 11. 前端组件拆分建议

```text
TenYearExamModal
├── ExamReveal
├── FinalRating
├── AxisScoreList
├── TurningPointList
└── ExamActions
```

组件只消费已有 `SectRating`：

```ts
interface SectRating {
  rank: "覆灭" | "勉强立足" | "小有名望" | "一方大宗" | "仙门新星";
  totalScore: number;
  axisScores: Record<"人" | "财" | "物" | "势", number>;
  summary: string;
  turningPoints: string[];
}
```

首版不要求 AI 额外生成复盘内容，避免弹窗依赖第三次 AI 调用。

## 12. 验收标准

1. 第十年结算后自动弹出，且年报和存档已经保存。
2. 第一眼能识别最终评级和总分。
3. 展开后能在 10 秒内看懂最强轴、最弱轴和三个关键转折。
4. 1280×720、1440×900、390×844 下无文字遮挡、按钮溢出或页面错位。
5. 弹窗内容不超出视口，长内容可内部滚动。
6. 背景主界面不可误触。
7. 关闭后可在宗门史书重新查看。
8. 继续执掌不会覆盖大考结果。
9. 重开十年有二次确认。
10. 所有美术资源不包含写死的动态文字和分数。

## 13. 首版边界

本次只制作大考评分弹窗，不包含完整十年 AI 复盘报告。

后续复盘报告可继续扩展：

- 十年最常用姿态。
- 初始与最终数值对比。
- 最大收益、最大代价和最大风险。
- AI 润色的史官总评。
- 下一轮经营建议。
